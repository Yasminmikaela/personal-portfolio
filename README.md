# Personal Portfolio — Yasmin Haanpää

A personal portfolio built as part of my ICT studies, with a focus on UX/UI design and frontend development.

🔗 **Live:** coming soon

---

## Built With

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/)

## Features

- Responsive design — works on mobile, tablet and desktop
- Smooth scroll navigation
- Animated UI elements
- Dark theme with custom color palette

## Getting Started

```
# Install dependencies
npm install

# Start development server
npm run dev
```

## Project Structure

```
src/
  assets/        # Images and static files
  components/    # Reusable UI components
  layout/        # Navbar and layout components
  sections/      # Page sections (Hero, About, Projects...)
```

## About

This portfolio is also my thesis project, exploring UX/UI design principles through hands-on implementation. The goal was to combine clean code with intuitive design to create a meaningful digital experience.
